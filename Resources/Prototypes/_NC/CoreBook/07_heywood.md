# Глава 7: Хейвуд (Heywood)

Хейвуд — это одновременно "спальня" города и его административный мозг. Здесь переплетаются политика, промышленность и латиноамериканские корни.

## 1. Уэллспрингс (Wellsprings / Heywood Docks)
Промышленная и портовая зона, критически важная для экономики Найт-Сити.
- **Суть:** "Артерия" снабжения. Порт принимает грузы со всего мира, несмотря на логистический кризис.
- **Контроль:** Разделен между корпорациями, профсоюзами докеров и бандами, которые "крышуют" перевозки.
- **Уровень угрозы:** Security Code 8 (High/Industrial Security). Территория под жестким надзором частных армий.

## 2. Глен (The Glen)
Политический и административный центр Найт-Сити 2045 года.
- **Суть:** Здесь находится мэрия (City Hall), штаб-квартиры крупных корпораций и главное управление NCPD.
- **Атмосфера:** Деловой район, где бюрократическая власть встречается с корпоративными интригами.
- **Уровень угрозы:** Security Code 7. Высокая концентрация сил правопорядка.

## 3. Виста-дель-Рей (Vista del Rey)
Более бедная и опасная часть Хейвуда, граничащая с Combat Zone.
- **Банды:** Традиционная территория **Valentinos**. Здесь сильны семейные узы и верность району.

## Значение для механик (Night City Station)
- **Фракция NCPD:** Глен — стартовая точка или хаб для игроков-полицейских.
- **Контрабанда:** Порт Уэллспрингс — идеальное место для миссий по перехвату или защите ценных грузов.
- **Политический геймплей:** Взаимодействие с мэрией и корпоративными юристами для получения лицензий или разрешений.
